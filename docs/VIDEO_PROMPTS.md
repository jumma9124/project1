# AI 영상 프롬프트 리스트

> 28개 금융 용어 전체에 대해 '일상 공감 및 시뮬레이션' 컨셉
> 사회초년생이 겪을 법한 현실적인 상황들을 따뜻한 파스텔 톤 웹툰 스타일로 구성

---

## 카테고리 1: 월급과 세금 (1~5번)

### 1. 세전/세후
연봉계약서의 큰 숫자와 실제 입금된 작은 숫자를 비교하며 갸우뚱하는 주인공.

```
Warm, pastel webtoon style. A character looks at a glowing contract showing a large amount. Then the screen shows a bank app notification with a smaller amount. A sign language interpreter in the top right box explains the gap.
```

### 2. 실수령액
월급 뭉치가 여러 세금 허들을 넘어 작아진 채 주인공 품에 안기는 경주.

```
A stack of cash running on a track, jumping over hurdles labeled 'Tax' and 'Insurance'. It gets smaller with each jump, and the character catches it at the finish line. Sign language guide in the top right.
```

### 3. 연말정산
밤늦게 영수증들을 모아 마법 상자에 넣고 '세금 환급' 선물을 기다리는 설렘.

```
A character putting digital receipts into a magic mailbox. The mailbox glows and produces a 'Refund' gift box. Cozy lamp lighting, pastel tones. Sign language interpreter in the top right.
```

### 4. 소득공제/세액공제
무거운 세금 배낭을 메고 가는데 '공제' 풍선들이 나타나 배낭을 가볍게 들어줌.

```
A character carrying a heavy backpack labeled 'Tax'. Two colorful balloons labeled 'Deduction' and 'Credit' lift the backpack, making the character walk lighter. Sign language guide in the top right.
```

### 5. 고정지출/변동지출
달력에 매달 찍히는 규칙적인 도장과 갑자기 튀어나오는 치킨/쇼핑 이모지들의 추격전.

```
A digital calendar with regular blue stamps appearing. Suddenly, colorful icons like 'Food' and 'Shopping' pop up randomly. The character tries to catch them. Sign language interpreter in the top right.
```

## 카테고리 2: 은행과 저축 (6~11번)

### 6. 예금 vs 적금
큰 돈을 한 번에 넣고 잠그는 금고와 매달 한 칸씩 쌓아 올리는 계단식 저금통.

```
Split screen. Left: Character puts a huge coin bag in a vault. Right: Character adds one small coin every month to build a staircase. Sign language guide in the top right.
```

### 7. 단리 vs 복리
매달 사과 한 개를 주는 나무와, 사과가 다시 나무가 되어 숲을 이루는 과정.

```
Left: A tree giving one apple every season. Right: An apple falls and grows into a new tree, creating a forest of apples. Sign language interpreter in the top right.
```

### 8. 기준금리
거대한 황금 수도꼭지를 돌리자 동네 은행들의 작은 분수 높이가 함께 변함.

```
A giant golden faucet labeled 'Base Rate'. Turning it up makes all the smaller fountains labeled 'Bank Interest' rise higher. Sign language guide in the top right.
```

### 9. 비과세
세금 새들이 주변을 맴돌지만, 투명한 유리 돔 속에 담긴 내 돈은 건드리지 못함.

```
Shiny coins inside a glowing glass dome. Small 'Tax Birds' try to peck at them but fail. The character smiles. Sign language interpreter in the top right.
```

### 10. ISA
여러 간식을 한 봉지에 담아 할인을 받는 '종합 선물 세트' 주머니.

```
A character puts 'Savings', 'Stocks', and 'Funds' into one big 'ISA' magic bag. The bag glows with a 'Tax Benefit' ribbon. Sign language guide in the top right.
```

### 11. CMA/파킹통장
잠시 주차해도 이자가 쑥쑥 자라는 편안한 '돈 전용 주차장'.

```
A cute car made of coins parks in a glowing 'Parking Account' space. Every second, small sparks of interest pop up around it. Sign language interpreter in the top right.
```

## 카테고리 3: 신용과 대출 (12~17번)

### 12. 신용점수
도서관에서 책을 잘 반납할 때마다 머리 위 별 점수가 올라가는 신뢰 게임.

```
A character returns books on time and gets shiny stars. A score bar above the head grows and turns gold. Sign language guide in the top right.
```

### 13. 원금
처음 빌려왔던 덩어리 그대로의 순수한 은빛 블록.

```
A clean, large silver block labeled 'Principal'. No extra attachments. Very simple and clear design. Sign language interpreter in the top right.
```

### 14. 원리금
은빛 블록(원금)에 '이자'라는 이름의 작은 구슬들이 줄줄이 매달려 있는 모습.

```
The silver block from before, but now with many small glowing beads labeled 'Interest' attached to it. Sign language guide in the top right.
```

### 15. 대출
부족한 사다리를 채우기 위해 미래의 내가 건네주는 황금 사다리 조각.

```
A character can't reach a goal. A future version of the character hands over a glowing ladder piece labeled 'Loan'. Sign language interpreter in the top right.
```

### 16. 마이너스 통장
지갑이 비었을 때만 나타나는 마법의 비밀 비상구 입구.

```
A character opens an empty wallet. A secret glowing door appears inside, offering a helping hand with coins. Sign language guide in the top right.
```

### 17. DSR
배낭에 빚 뭉치를 넣을 때, 너무 무거우면 가방 끈이 빨갛게 변하며 알려주는 센서.

```
A character putting 'Debt' weights into a backpack. If it's too much, the backpack straps flash a warning red light. Sign language interpreter in the top right.
```

## 카테고리 4: 투자 기초 (18~21번)

### 18. 주식
내가 좋아하는 브랜드의 로고가 새겨진 화분을 정성껏 가꾸는 과정.

```
A character planting a seedling with a famous company logo. It grows with the character's care. Sign language guide in the top right.
```

### 19. 채권
친구에게 돈을 빌려주고 정해진 날짜에 이자 쿠폰을 받는 약속 카드.

```
A character hands coins to a building and gets a 'Promise Scroll'. The scroll gives small interest coins regularly. Sign language interpreter in the top right.
```

### 20. ETF
사과, 포도, 바나나를 낱개로 사기 힘들 때 한 번에 산 '과일 컵' 세트.

```
A character picks up a pre-mixed fruit cup labeled 'ETF' instead of buying whole crates of different fruits. Sign language guide in the top right.
```

### 21. 배당금
나무가 잘 자라준 덕분에 주인에게 보답으로 떨어지는 보너스 황금 열매.

```
A healthy tree suddenly drops a few gold coins into the character's hands as a surprise gift. Sign language interpreter in the top right.
```

## 카테고리 5: 주거와 부동산 (22~26번)

### 22. 전세 vs 월세
커다란 보증금 뭉치를 맡기거나, 매달 작은 동전을 건네며 열쇠를 지키는 모습.

```
One side: Giving a huge coin bag to stay. Other side: Giving one coin every month to stay. Both hold house keys. Sign language guide in the top right.
```

### 23. 등기부등본
돋보기를 들고 집의 신분증을 꼼꼼히 읽으며 안심하는 주인공.

```
A character looking through a magnifying glass at a house's ID card. It shows 'Safe' with a green checkmark. Sign language interpreter in the top right.
```

### 24. 확정일자
계약서에 황금 도장을 쾅 찍자 집 주변에 튼튼한 방어막이 생기는 연출.

```
A big golden stamp hits a paper. A glowing shield forms around the character's coin bag and home. Sign language guide in the top right.
```

### 25. 주택청약
내 집 마련이라는 높은 산을 향해 매달 한 계단씩 벽돌을 쌓는 정성.

```
A character adds one brick labeled 'Subscription' every month to build a path toward a dream house on a hill. Sign language interpreter in the top right.
```

### 26. 임대차계약서
집주인과 마주 앉아 '서로 배려하며 살기'로 약속하는 훈훈한 장면.

```
A character and a landlord sign a glowing paper and shake hands. The paper turns into a warm light protecting the house. Sign language guide in the top right.
```

---

## 카테고리 6: 거시경제와 생활 (27~28번)

### 27. 인플레이션
어제는 1,000원에 샀던 빵이 오늘은 반쪽이 되어 있는 슬픈 마법.

```
A character holds a coin and looks at a bread. The bread suddenly shrinks while the coin stays the same. Sign language interpreter in the top right.
```

### 28. 공공요금
전기를 아껴 쓴 주인공에게 에너지 요정이 윙크하며 가벼운 고지서를 주는 장면.

```
A character turns off lights. An energy fairy smiles and leaves a small, light paper bill with a heart. Sign language guide in the top right.
```
